## Project Scope

The Inventory Tracker is a simple web-based application developed to manage and display inventory information. The project helps users keep track of products, quantities, and prices in an organized manner. It demonstrates the use of HTML, CSS, and SQL for creating a basic inventory management system.

---

## Technologies Used

- HTML
- CSS
- SQL

---

## Features

- Display inventory records in a structured table.
- Show Product ID, Product Name, Quantity, and Price.
- Simple and user-friendly interface.
- Database script included for inventory data management.
- Easy to understand and maintain.
