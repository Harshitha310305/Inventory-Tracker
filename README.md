# Inventory Tracker

## Description

A simple Inventory Tracker project built using MySQL.

## Features

* Add products
* View products
* Update product quantity
* Search products
* Delete products

## Technologies Used

* MySQL
* HTML
* CSS

## Database

Database Name: inventory_db

Table Name: products

## Setup

1. Import the database schema into MySQL (create database `inventory_db` and `products` table).
2. Update database connection details in the application configuration.
3. Run the web server and open the app in your browser.
